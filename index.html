I've created a small Golang webserver that executes a Linux command to generate a new user.

I used hwdsl2's script because its more versatile than angristan's.
Here is the basic command I would use to generate a new user using the WireGuard script.

$ ./wireguard.sh --addclient NEW_CLIENT_NAME --dns1 PIHOLE_DNS

This is how I setup PiHole to be DNS server, so it can filter out ads.

$ curl -sSL https://install.pi-hole.net | bash
# Pick the physical interface, i.e, eth0
# Pick cloudflare (doesn't matter)
# Accept the rest...

Then I made some changes to /etc/lighttpd/lighttpd.conf file, so it can listen to a different port, so it doesn't collide with the Golang server.

$ vim /etc/lighttpd/lighttpd.conf # port 80 -> 8020

After all that I had to enter the PiHole web admin interface and configure DNS to permit all origins, so my WireGuard can use PiHole as its DNS.
